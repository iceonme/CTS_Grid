# REFERENCE — Zen 7.1 参数详解

本文档提供 `config.json` 中每个参数的详细说明、推荐范围及回测结论。

---

## 参数详解

### 基础参数

| 参数 | 默认值 | 类型 | 说明 |
|---|---|---|---|
| `resample_min` | `60` | int | 将 1m K 线重采样为 n 分钟 K 线。60 = 1小时棒 |
| `capital` | `10000` | float | 策略运营总资金（USDT），用于计算每格仓位大小 |
| `grid_layers` | `5` | int | 最大网格层数，总资金平均分配到每一层 |

### 网格控制参数

| 参数 | 默认值 | 类型 | 说明 |
|---|---|---|---|
| `grid_drop_pct` | `0.02` | float | 触发补仓所需的跌幅比例。相对于上一层均价，价格需再跌 ≥ 2% 才允许加仓 |

**调参建议**：
- `0.015`（1.5%）：适合低波动期，加仓更积极
- `0.02`（2%）：标准值，回测 2025 年表现最稳定
- `0.03`（3%）：适合高波动期，减少摊薄频率

### 止盈止损参数

| 参数 | 默认值 | 类型 | 说明 |
|---|---|---|---|
| `hard_sl_pct` | `-0.10` | float | 全局硬止损线。整体浮亏达此阈值时，1m K 线实时触发全仓清出 |
| `tp_min_profit_pct` | `0.03` | float | 动态止盈保底盈利。整体盈利达此值且满足动态条件时触发止盈 |

**止盈触发逻辑（任一满足）**：
1. 盈利 ≥ `tp_min_profit_pct` AND 曾触上轨 AND RSI 曾 > 65 AND 当前价格 < 历史最高价
2. 盈利 ≥ `tp_min_profit_pct × 2`（无条件止盈）

## 指标参数（Z71Indicators 内置，不在 config.json 中暴露）

如需调整，需直接修改 `strategy.py` 中 `Z71Indicators` 的默认值：

| 参数 | 默认值 | 说明 |
|---|---|---|
| `boll_period` | `20` | 布林带计算周期 |
| `boll_std` | `2` | 布林带标准差倍数 |
| `macd_fast` | `12` | MACD 快线 EMA 周期 |
| `macd_slow` | `26` | MACD 慢线 EMA 周期 |
| `macd_sig` | `9` | MACD 信号线 EMA 周期 |
| `rsi_period` | `14` | RSI 计算周期 |
| `vol_ma_period` | `5` | 成交量均线周期 |

| 参数组合 | 总收益 | 最大回撤 | 胜率 | 备注 |
