8.5的策略核心

# 基础策略
1. 主要看1分钟K线，从当前时间向前看，过去6小时（360分钟），


2. 网格顶部与底部计算
分为5个时间段，每段约72分钟，每个时间段取1个高点 + 1个低点
- 原始5个高点：[H1, H2, H3, H4, H5] 按大小排序
- 去掉：最高1个（H5）+ 最低1个（H1）
- 保留：H2, H3, H4 
- 取平均值 = 网格顶部

原始5个低点：[L1, L2, L3, L4, L5] 按大小排序  
去掉：最低1个（L1）+ 最高1个（L5）
保留：L2, L3, L4 取平均 = 网格底部

3. 网格层数设计方式
- 波动率高则为7层
- 波动率低则为5层
每层高度由（顶部-底部）/层数

波动率计算
计算过去6小时的波动率
波动率 = (网格顶部 - 网格底部) / 网格底部

> 1.2% → 高波动（7层网格）
≤ 1.2% → 低波动（5层网格）

4. 交易模式
-5层模式 
-- 下中间层L(0) 为缓冲观望层，上下两层L(-2),L(-1),L(1),L(2)为实体交易层，再上下两层L(-3),L(-4),L(3),L(4)为虚拟层。
-- 当价格落入L(-1)或L(-2)时，买入，买入价格是该层的下半层的价格，当价格落入L(1)或L(2)时，卖出，卖出价格是该层的上半层的价格
-- 当价格落入虚拟层时需要根据RSI来判断是否买入或卖出,见下文说明
-7层模式 下中间层L(0)为缓冲观望层，上下三层L(-3),L(-2),L(-1),L(1),L(2),L(3)为实体交易层，再上下两层L(-4),L(-5),L(4),L(5)为虚拟层。
-- 交易方式与5层类似。

- 交易量与仓位
-- 每次买入都是当前资金的1/n,n是当前层数
-- 每次卖出都是当前仓位的1/n

- 动态RSI机制
RSI(14)用于辅助在虚拟层区间的交易辅助决策
-- 高波动率> 2%下，	RSI<20买入,	RSI>80卖出
-- 波动率在1.2% - 2%范围内，RSI<25买入	RSI>75卖出
-- 低波动率< 1.2%下，RSI<30买入, RSI>70卖出 

综上所述，希望实现在价格在网格内波动时 高抛低吸。


# 网格调整及熔断
如果价格突破超出虚拟层范围，则暂停两小时进入观察期，如观察期价格回到网格内，则恢复买入卖出操作。如超过2小时没有回到网格，则重新根据当前时间重新计算网格，进入新的网格阶段
