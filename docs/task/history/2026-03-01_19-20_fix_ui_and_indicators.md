# 任务验收文档: 布局修复与指标增强 (2026-03-01 19:20)

## 任务目标
解决 V5.1 仪表盘的布局遮挡问题，并根据策略核心逻辑补全关键指标（ATR、波段参考点）的实时显示。

## 已完成的改进
- **[√] MACD 时间轴对齐**: 修复了 MACD 指标向左缩进的问题。
- **[√] 后台刷新稳定性**: 引入 `Late-Join` 机制，解决页面刷新后历史数据消失的问题。
- **[√] 重置逻辑优化**: 重置操作现在会立即触发 `reset_ui` 并同步策略停止状态到前端按钮。
- **[√] ATR 实时计算**: 修复了策略停止时 ATR 显示为 0 的问题，现在无论是否启动都会实时计算。
- **[√] UI 布局滚动修复**: 移除了针对 `body` 和 `main` 的强制 `overflow: hidden` 限制，允许页面根据内容自动增长，左侧底部指标不再被遮挡。
- **[√] 策略指标增强**:
  - 增加了 **波段参考点 (Top 3 High/Low)**：即策略逻辑中用于构建动态网格的 3 高 3 低阻力支撑位文字展示。
  - 补全了 **可用资金、RSI 阈值、信号强度、交易信号** 等 10 多个数据字段的实时绑定。
