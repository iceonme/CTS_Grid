# V85 策略逻辑修复验收报告 (Walkthrough)

**日期**：2026-03-09
**版本**：V8.5 (Bug Fixed)
**目标**：全面修复网格计算、指标计算及买卖逻辑中的重大缺陷。

## 1. 修复内容清单

### 🔴 核心逻辑纠正
- **网格层级重映射**：
    - 修复了 `grid_lines` 分布问题。不再是简单均分，而是严格按照 `base_bottom` 到 `base_top` 划分为 $n$ 个实体层（5或7层）。
    - 重新计算了 `l0_idx`，确保 **L0 缓冲区** 始终位于网格正中心，并在买卖方向判定之间起到隔离作用。
- **资金管理对齐文档**：
    - **买入**：改为动态计算，每次成交金额定为 `当前总资产 * max_position_pct / n`。
    - **卖出**：改为分批出场，每次卖出 `当前持仓总量 / n`。
- **RSI 计算标准化**：
    - 弃用简单移动均值 (SMA)，改用标准 Wilder's Smoothing (EMA)，以提高对价格波动的响应灵敏度，优化虚拟层买卖时机。
- **熔断状态重置**：
    - 网格每 6 小时重算或熔断重算时，自动清空 `layer_holdings`。防止旧网格的层级锁定延续到新网格导致逻辑死锁。
