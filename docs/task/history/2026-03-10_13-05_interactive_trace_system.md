# 交互式决策追踪系统 (Interactive Trace) 验收报告

**日期**: 2026-03-10 13:10
**描述**: 将 V8.5 策略的决策追踪系统由静态快照升级为**全量时间轴映射系统**，并实现了 Dashboard 的光标联动交互。

## 升级核心功能

### 1. 全量逻辑映射
- **存储优化**: 策略层 `decision_trace` 升级为以毫秒时间戳为键的字典。
- **全量存档**: 记录回测期间每一分钟的所有逻辑分支代码活动，不再限制为最近 20 条。
- **数据透亮**: 回测结果 JSON 包含每一分钟的“策略内心活动”，为深度审计提供 100% 数据支持。

### 2. Dashboard 光标联动交互 (指哪看哪)
- **实时刷新**: 鼠标在 K 线图上移动时，右侧日志面板会自动刷新为对应时刻的策略判断逻辑。
- **高亮展示**: 下单成交信息以蓝色高亮显示。
- **操作逻辑**: 用户不再需要翻阅滚动条，由于数据已对齐，直接指点任意 K 线即可回访当时策略的情况。

## 验证与测试
- **逻辑测试**: 运行 `test_v85_logic.py` 确认字典化后的记录逻辑依然精准。
- **性能测试**: 全量日志（单日约 1440 个点）对 JSON 加载和浏览器渲染几乎无感知。
- **端到端流程**: 运行 `run_v85_static_viewer.py` -> 刷新 Dashboard -> 移动光标 -> 日志实时变化。验证成功。
